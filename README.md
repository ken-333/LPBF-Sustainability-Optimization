# Sustainability-oriented Multi-objective Optimization for High Layer Thickness LPBF

This repository contains the data and code for the paper: **"Laser Powder Bed Fusion with high layer thickness: sustainability oriented multi-objective optimization"**.

## Overview
This study proposes a two-stage optimization model to balance **Energy Efficiency**, **Carbon Emissions**, and **Cost-effectiveness** in LPBF processing of SS-CX die steel.

**Key Methodologies:**
* **Design of Experiment:** Doehlert Design (84 sets of parameters).
* **Modelling:** Polynomial Regression for Relative Density (RD).
* **Optimization:** AUGMECON-R algorithm (Exact Pareto front generation).
